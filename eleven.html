CSS Positioning Techniques

Static Positioning

This is a static positioned element. It is the default positioning type in CSS.

Relative Positioning

This element is positioned relative to its normal position. Notice how it moves without affecting other elements.

Absolute Positioning

This element is positioned absolutely within its nearest positioned ancestor (relative/absolute). It does not affect the layout of other elements.

Fixed Positioning

This element is fixed to the viewport, so it stays in the same position when scrolling.

Sticky Positioning

This element is sticky. It behaves like relative until it crosses a specified point, then it behaves like fixed.
